1. What is a JSP Expression?

A JSP Expression is used to evaluate a Java expression, convert into a String, and display the result each time the page is requested.

2. Syntax

<%= Java Expression %>

This is different from a Scriptlet in that it contains only a single expression, and converts the expression into a string, and includes it directly to the output stream.

Scriptlet: 

    <% 
        String hellowWorld = "Hellow World";
        System.out.println(hellowWorld);
    %>
Expression: 
    <%= hellowWorld %>
