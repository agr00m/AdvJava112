Analyzer:
$ ant compile_analyzer      Compile analyzer project
$ ant build_analyzer		Compile & build analyzer project
$ ant run_tests_analyzer    Runs the analyzer tests
$ ./runAnalyzer.sh ***      Runs the analyzer project

Labs:
$ ant compile_labs		    Compile all labs
$ ant build_labs            Compile & build all labs
$ ant run_tests_lab#        Runs the specified lab tests
$ ./runLab***.sh            Runs the specificed lab

Demos:
$ ant compile_demos         Compile everything in the demo directory
$ ant build_demos           Compile and build demos
$ ant compile_analyzer		Compile analyzer project
